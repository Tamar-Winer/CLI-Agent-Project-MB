# 🚀 CLI Command Station - Windows Agent
**מפתח:** Tamar Winer  
**טכנולוגיה:** Python | Gemini 2.5 Flash | Gradio

סוכן בינה מלאכותית מאובטח המתרגם פקודות בשפה חופשית (עברית/אנגלית) לפקודות ברות-ביצוע ב-Windows Command Line (CMD).

## ✨ תכונות עיקריות
- **תרגום חכם:** שימוש במודל Gemini 2.5 Flash לתרגום מדויק של בקשות מורכבות.
- **פרוטוקול אבטחה:** מנגנון מובנה לזיהוי וחסימת פקודות מסוכנות (כמו פורמט או מחיקות המוניות).
- **הגנת הזרקה:** חסינות לניסיונות שינוי זהות המודל (Prompt Injection).
- **ממשק UI יוקרתי:** עיצוב Dark Mode ממורכז המבוסס על Gradio עם התאמות CSS אישיות.

## 🛠 דרישות קדם
- Python 3.8+
- מפתח API מ-Google AI Studio.
